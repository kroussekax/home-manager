# Home-Manager

> This is my personal NixOS HM configuration with minimal packages, optimal configurations, and clean code (or at least i try)

## Technologies Used

[HomeManager](https://nix-community.github.io/home-manager/) -> Main tool used

- [Vesktop](https://vencord.dev) → Custom Discord client

- [Kitty](https://sw.kovidgoyal.net/kitty/) → GPU-accelerated terminal

- [Git](https://git-scm.com) → Version control

- [PrismLauncher](https://prismlauncher.org) → Minecraft launcher

- [Tmux](https://github.com/tmux/tmux) → Terminal multiplexer

- [Zsh](https://zsh.org) → Shell

- [PavuControl](https://freedesktop.org/software/pulseaudio/pavucontrol/) → Audio volume control

- [Ripgrep](https://github.com/BurntSushi/ripgrep) → Fast grep

- [Unzip](https://infozip.sourceforge.net) → ZIP extractor

- [Wget](https://gnu.org/software/wget/) → File downloader

- [Yad](https://github.com/v1cont/yad) → GUI dialog tool

- [Fzf](https://github.com/junegunn/fzf) → Fuzzy finder

- [Fastfetch](https://github.com/fastfetch-cli/fastfetch) → System info fetch

- [notify-desktop](https://github.com/nowrep/notify-desktop) → Desktop notifications

- [xsettingsd](https://github.com/derat/xsettingsd) → X settings daemon

- [Node.js](https://nodejs.org) → JavaScript runtime

- [GCC](https://gcc.gnu.org) → C/C++ compiler

- [CMake](https://cmake.org) → Build system generator

- [Ninja](https://ninja-build.org) → Fast build system

- [Lua](https://lua.org) → Scripting language

- [LuaRocks](https://luarocks.org) → Lua package manager

- [Go](https://go.dev) → Go language

- [Rustup](https://rustup.rs) → Rust toolchain manager

- [Python 3](https://python.org) → Python language

- [Zig](https://ziglang.org) → Zig language

- [Gawk](https://gnu.org/software/gawk/) → AWK implementation

- [JetBrains Mono Nerd Font](https://github.com/ryanoasis/nerd-fonts) → Patched monospace font

- [Gnome Themes Extra](https://gitlab.gnome.org/GNOME/gnome-themes-extra) → GTK themes

- [Papirus Icon Theme](https://github.com/PapirusDevelopmentTeam/papirus-icon-theme) → Icon theme
